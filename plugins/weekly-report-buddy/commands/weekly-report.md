# 周报生成

请按以下步骤为用户生成本周周报：

## 第一步：获取本周数据

分别执行以下命令拉取飞书数据（注意时间范围为本周一 00:00 到当前时间）：

1. **日历日程**：
   ```
   lark-cli calendar +agenda --start "<本周一T00:00:00+08:00>" --end "<当前时间>"
   ```

2. **任务列表**：
   ```
   lark-cli task +list
   ```

3. **本周文档产出**（搜索用户近期编辑的文档）：
   ```
   lark-cli docs +search --query "" --page-size 20
   ```

## 第二步：整理周报内容

将获取的数据整理为以下结构：

### 📋 本周完成
- 根据已完成的任务和已过去的日程，列出关键成果
- 每条用一句话概括，附带具体数据或产出物

### 🔄 进行中
- 根据未完成的任务和未来几天的日程，列出正在推进的事项
- 标注当前进度和预计完成时间

### 📅 下周计划
- 根据下周日历日程和待办任务，列出下周重点事项
- 按优先级排列

### 📄 本周文档产出
- 列出本周创建或编辑的飞书文档（标题 + 链接）

## 第三步：输出为飞书文档

将周报内容通过飞书 CLI 创建为飞书文档：

```
lark-cli docs +create --title "周报 - <用户名> - <本周日期范围>" --markdown "<生成的周报 Markdown 内容>"
```

创建成功后，输出文档链接给用户。
